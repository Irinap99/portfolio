Matière

Crafted as part of my UI/UX practice in Figma, this project explores the creation of an editorial-style eCommerce experience. The focus was on balancing brand storytelling with functional shopping flows, while keeping the design responsive and user-friendly.

See full case study

Design Process: I studied minimalist design trends and applied them to create a refined yet approachable visual system. My process included research, wireframing, high-fidelity mockups, and responsive adaptations for both desktop and mobile. Particular attention was given to typography, color harmony, and product photography integration.

Challenges: One of the main challenges was maintaining a balance between editorial storytelling and usability. I also had to ensure that mobile browsing felt intuitive without losing key brand elements solved through horizontal scrolling sections and simplified navigation. Another challenge was optimizing checkout flow to minimize friction while still feeling cohesive with the brand aesthetic.

Technologies Used: Figma for wireframing, prototyping, and building the design system. Photoshop for mockups, image editing.

User Goals & Flows I identified the main user goals as: discovering products, exploring collections, and completing a purchase with minimal friction. To support these, I mapped core flows such as browsing, product discovery, cart management, and checkout. This ensured that the design balanced editorial storytelling with clear, accessible shopping pathways across both desktop and mobile.

Features: Editorial-style landing page combining storytelling with product highlights, streamlined checkout with minimal steps to reduce friction, responsive layouts for desktop and mobile without loss of functionality, warm tactile color palette with neutral tones supporting product photography, clear visual hierarchy using Garamond for storytelling elements and Open Sans for product-related content.

Future Improvements: Next steps for this project would include conducting user testing to validate the checkout flow and mobile navigation, as well as refining micro-interactions such as hover states and cart transitions to add subtle delight. I'd also expand filtering and personalization options to simulate scalability and create a more tailored shopping experience.
