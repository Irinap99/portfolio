natours

Crafted with the guidance of my instructor, this project exemplifies the utilization of float-based layouts — a fundamental technique in traditional web development. Despite the emergence of more modern layout technologies, understanding float properties remains crucial for projects that rely on legacy codebases.

Visit website

Design Process: I gained a profound appreciation for the significance of color theory in conveying messages effectively. Understanding the psychology behind color usage became essential in communicating the intended brand identity and message. Each color choice was meticulously considered, ensuring alignment with the project's overarching theme and goals.

Challenges: Navigating the use of floats in Natours presented a unique challenge, particularly in a landscape dominated by more modern layout technologies. While floats may appear outdated compared to contemporary tools, harnessing them for a modern website was an enlightening experience. Despite the inherent limitations of floats, their utilization in Natours underscored the importance of versatility and resourcefulness in web development.

Technologies Used: From float properties to clip-path tools, I learned diverse techniques to shape elements and achieve desired layouts. Additionally, I implemented engaging animations on page load and utilized videos as backgrounds to enhance user experience as well as a card flipping effect on hover, adding a dynamic touch to the user interface. Building functional forms further solidified my skills in user interaction design. Natours served as a comprehensive learning experience, showcasing my ability to integrate various technologies effectively for impactful web development.

Features: Trillo incorporates user reviews, an interactive search bar, a user menu, an inbox with real-time notifications, a dynamic call-to-action button, and a sidebar with engaging animations to enhance user experience and interaction.
